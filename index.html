HTML5 Introduction

    HTML5 Introduction

  • HTML5 is the latest revised version of HTML, which was accomplished by W3C in October,2014.
  • The purpose of the design of HTML5 is to support multi-media in mobile devices.
  • HTML5 is very easy to learn.

    What HTML5?

  • HTML, the last version of HTML4.01 waspublished in 1999.From then on the world of Web had greatly changed.
  • The purpose of the design of HTML5 is to support multi-media in mobile devices.
  • HTML5 is still in development.But most of mordern browsers have already supported HTML5.

    What is the beginning of HTML5?

  • HTML5 is the result of the cooperation of W3C and WHATWG(Web Hypertext Application Technology Working Group.
  • WHATWG dedicate to on web tables and applications,and W3C focus on XHTML2.0.In 2006,these two groups decided to work together to create a new version of HTML.

    • Some interesting features in HTML5:

  • Canvas which is used to paint.
  • Video and audio which are used to media playback.
  • Support more on local offline storage.
  • Brand new content-control element such as article,footer,header,nav,section.
  • New table and form controlling element such as calender,date,time,email,url,search.
